📊 Power BI Learning Hub

Power BI is Microsoft's Business Intelligence tool used to create dashboards, reports and business insights.


1. Data Import

Import data from Excel, CSV, SQL Server and other sources.

2. Power Query

Used for data cleaning and transformation.

  • Remove Duplicates
  • Change Data Types
  • Split Columns
  • Merge Queries

3. Data Modeling

Create relationships between tables.

  • One-to-Many
  • Many-to-One
  • Star Schema

4. DAX

DAX (Data Analysis Expressions) is used for calculations. 

Total Sales =
SUM(Sales[Amount])

5. Dashboard Design

  • KPI Cards
  • Bar Charts
  • Line Charts
  • Slicers
  • Maps
