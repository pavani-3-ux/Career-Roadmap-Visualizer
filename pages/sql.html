🗄 SQL Learning Hub

SQL (Structured Query Language) is used to store, manage and analyze data in databases. Every Data Analyst must know SQL.


1. SELECT Statement

Used to retrieve data from a table. 

SELECT * FROM Employees;

2. WHERE Clause

Used to filter records. 

SELECT * FROM Employees
WHERE Salary > 50000;

3. GROUP BY

Used to group rows with similar values. 

SELECT Department,
COUNT(*)
FROM Employees
GROUP BY Department;

4. JOINS

Used to combine data from multiple tables. 

SELECT *
FROM Orders
JOIN Customers
ON Orders.CustomerID =
Customers.CustomerID;
