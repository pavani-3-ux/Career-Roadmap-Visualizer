📈 Statistics Learning Hub

Statistics helps Data Analysts understand data, identify patterns, and make data-driven decisions.


1. Mean

The average value of a dataset. 

Mean = Sum of Values / Number of Values

2. Median

The middle value in a sorted dataset.

3. Mode

The most frequently occurring value.

4. Probability

Measures the chance of an event occurring. 

Probability =
Favorable Outcomes / Total Outcomes

5. Standard Deviation

Shows how spread out the data is from the mean.

6. Hypothesis Testing

Used to test assumptions using sample data.

  • Null Hypothesis (H0)
  • Alternative Hypothesis (H1)
  • P-Value
