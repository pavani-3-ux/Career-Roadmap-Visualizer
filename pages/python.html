🐍 Python Learning Hub

Python is one of the most important programming languages for Data Analysts. It is used for data cleaning, analysis, visualization and automation.


1. Variables

Variables store data values.

name = "Pavani"
age = 22

2. Loops

Loops help repeat tasks.

for i in range(5):
    print(i)

3. Functions

Functions help organize code.

def greet():
    print("Hello")

greet()

4. Pandas

Pandas is used for data analysis and data cleaning. 

import pandas as pd

df = pd.read_csv("sales.csv")
print(df.head())

5. Matplotlib

Matplotlib is used for data visualization. 

import matplotlib.pyplot as plt

plt.plot([1,2,3],[10,20,30])
plt.show()
