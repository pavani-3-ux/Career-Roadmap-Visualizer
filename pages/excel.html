📗 Excel Learning Hub

Microsoft Excel is one of the most important tools for Data Analysts. It is used for data cleaning, calculations, reporting and dashboards.


1. Basic Formulas

Excel formulas help perform calculations. 

=SUM(A1:A10)

=AVERAGE(A1:A10)

=MAX(A1:A10)

2. IF Function

Used for logical conditions. 

=IF(A1>100,"High","Low")

3. VLOOKUP / XLOOKUP

Used to find values from another table. 

=VLOOKUP(A2,D:G,2,FALSE)

4. Pivot Tables

Pivot Tables summarize large datasets quickly.

  • Sales Analysis
  • Department Summary
  • Regional Reports

5. Charts & Dashboards

  • Bar Chart
  • Line Chart
  • Pie Chart
  • Interactive Dashboard
