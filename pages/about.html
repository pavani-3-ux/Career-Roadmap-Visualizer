🚀 About Career Roadmap Visualizer

Career Roadmap Visualizer is a complete learning platform designed for students and aspiring professionals who want to build a successful career in Data Analytics and related fields.

The platform provides structured learning paths, skill tracking, progress monitoring, learning resources, certifications guidance and career recommendations.

Users can explore detailed roadmaps for Excel, SQL, Python, Statistics and Power BI while tracking their readiness score through the interactive dashboard.

Key Features:

  • 📊 Data Analyst Career Roadmap
  • 📈 Progress Tracking Dashboard
  • 🏆 Certification Recommendations
  • 📚 Learning Resources Hub
  • 🤖 AI Career Suggestions
  • 🎯 Career Quiz System
  • 🌳 Career Tree Visualization
  • 📱 Responsive Design

This project was built using HTML, CSS and JavaScript to help learners become job-ready through structured and guided learning.
